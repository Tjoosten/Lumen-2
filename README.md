## Lumen API

Lumen APi is een gebouwd op het Lumen framework van laravel.

## Documentation

The API has its own cli commands, the costum commands are:

| Command: | Description: |
| :------- | :---------- |
| `api:count` | Display and count all the database records related to the api |
